# MCP Templates Pack - 使用案例

## 案例 1: 自动化内容创作者

**用户画像**: 博主、YouTuber、播客制作人

**工作流**:
```
mcp-obsidian → 读取笔记草稿
     ↓
mcp-websearch → 搜索相关素材和数据
     ↓
mcp-n8n → 触发内容发布工作流
     ↓
mcp-fileops → 保存最终内容
```

**价值**: 节省 80% 的内容研究时间
